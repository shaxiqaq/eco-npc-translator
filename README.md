# eco-npc-translator

实时把 **ECO（Emil Chronicle Online）** 的 NPC 对话与选项菜单翻译成中文，**直接显示在游戏原生对话框里**——不闪屏、不重叠、不开覆盖层。

原理：用 Frida 在客户端进程内挂钩网络收包，当场解密封包 → 把英文替换成中文 → 重新加密写回，由游戏自己渲染。

## 特性

- 🎯 对话（op1017）+ 选项菜单（op1526）首屏即中文
- 🔁 `F9` 一键在 中文 / 英文原文 之间切换
- ⚡ 译文磁盘缓存，见过的对话秒开
- 🧩 图形化配置：下拉选服务商（DeepSeek / OpenAI / OpenRouter / Gemini / 本地 Ollama / DeepL），填入自己的 API Key
- 📦 翻译引擎已内置打包，解压即用

## 快速开始

```bash
pip install frida keyboard openai
```

1. 启动并登录游戏 `eco.exe`
2. 双击 `启动NPC翻译.cmd`（首次会自动弹出配置窗口）
3. 选服务商、填 API Key、保存，重新启动即可

详见 **[分发说明.md](分发说明.md)**。

## ⚠️ 客户端 / 服务器适配

`_mitm.js` 中的 AES 收割地址 `0x18cc4` 与 `eco_npc_mitm.py` 中的地图端口 `12002` 与**特定客户端版本 + 特定服务器**绑定。换客户端版本/服务器需自行重新定位（见分发说明）。
