# ECO NPC 实时翻译 — 使用 / 分发说明

让 ECO（Emil Chronicle Online）的 NPC 对话和选项菜单，**在游戏原生对话框里直接显示中文**。
原理：进程内挂钩网络收包，当场解密 → 把英文替换成中文 → 重新加密写回，由游戏自己渲染。
不改游戏文件、不开覆盖层，所以不闪、不重叠。

---

## 一、环境要求

| 项 | 要求 |
|---|---|
| 系统 | Windows |
| Python | 3.8 以上（命令行 `python --version` 能输出版本） |
| 依赖包 | 一行命令安装：`pip install frida keyboard openai` |
| 翻译服务 | DeepSeek / OpenAI / OpenRouter / Gemini / 本地 Ollama / DeepL 任选其一（需要对应的 API Key；Ollama 免费离线无需 key） |
| 游戏 | eco.exe 需先运行并已登录进地图 |

> Gemini 额外需要：`pip install google-genai`

---

## 二、首次使用

1. **先启动游戏 eco.exe**，登录到有 NPC 的地图。
2. 双击 **`启动NPC翻译.cmd`**。
   - 首次运行没有配置时，会**自动弹出配置窗口**。
3. 在配置窗口里：
   - **翻译服务商**：下拉选择（选完会自动填好接口地址和常用模型）。
   - **模型**：可下拉选常用的，也可手动输入任意模型名。
   - **API Key**：填你自己的 key（旁边「显示」可明文查看）。
   - 点 **测试连接** 确认能用（会试译一句 Hello）。
   - 点 **保存**。
4. 重新双击 **`启动NPC翻译.cmd`**，看到 `引擎预热完成` 即就绪。
5. 进游戏和 NPC 对话，即显示中文。

之后想换服务商 / 模型 / key，随时双击 **`配置翻译.cmd`** 修改并保存，重启翻译生效。

---

## 三、日常使用

- **启动**：双击 `启动NPC翻译.cmd`（保持那个黑窗口开着）。
- **停止**：关闭黑窗口。
- **切换中文 / 英文原文**：游戏里随时按 **F9**。
  - 黑窗口会打印 `[切换] 当前显示: 中文 / 英文原文`。
  - 若 F9 没反应，多为权限问题——右键用**管理员身份**运行 `启动NPC翻译.cmd`。
- **缓存**：见过的对话会存进 `npc_cache.json`，下次秒开。
- **首次见到的新对话**：会卡 1～3 秒等翻译（首屏即中文），属正常。

## 五、配置文件格式（手动编辑可选）

`translate_config.json`：

```json
{
  "provider": "deepseek",
  "model": "deepseek-v4-flash",
  "base_url": "https://api.deepseek.com",
  "api_key": "你的key"
}
```

各服务商参考：

| provider | base_url | 模型示例 | 是否需要 key |
|---|---|---|---|
| deepseek | https://api.deepseek.com | deepseek-v4-flash | 是 |
| openai | （留空） | gpt-4o-mini | 是 |
| openrouter | https://openrouter.ai/api/v1 | google/gemini-flash-1.5 | 是 |
| gemini | （留空） | gemini-2.0-flash | 是 |
| ollama | http://127.0.0.1:11434 | gemma4:12b | 否（需先 `ollama serve` + `ollama pull`） |
| deepl | https://api-free.deepl.com/v2 | （留空） | 是 |

## 六、⚠️ 客户端 / 服务器适配（重要）

本工具有两处与**特定客户端版本 + 特定服务器**绑定，换环境若对不上则无法工作：

1. **AES 收割地址**：`_mitm.js` 中 `m.base.add(0x18cc4)` 是当前 eco.exe 版本里加密函数的偏移。
   客户端大版本更新后偏移可能改变，需用逆向工具重新定位加密函数入口。
2. **地图端口**：`eco_npc_mitm.py` 中 `MAP_PORT = 12002`。换服务器若地图端口不同需对应修改。

只要是**同一个客户端版本 + 同一个服务器**，这两项无需改动，装好依赖、配好 key 即可使用。

## 七、常见问题

- **黑窗口提示「没有运行中的 eco.exe」**：先启动并登录游戏，再启动翻译。
- **一直显示英文不变中文**：检查黑窗口有没有报 key 错误；用 `配置翻译.cmd` 点「测试连接」确认 key 可用；确认地图端口正确。
- **某些行文字偏长溢出**：把 `eco_npc_mitm.py` 里 `wrap_cjk(zh, 20)` 的 `20` 调小（如 18）。
- **新对话等待太久**：`eco_npc_mitm.py` 里 `SYNC_TIMEOUT = 4.0` 可调大（网络慢时），但太大可能让收包线程阻塞过久。
